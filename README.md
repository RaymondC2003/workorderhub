# WorkOrderHub

![WorkOrderHub](frontend/public/logo.png)

## Run

**macOS / Linux**
```bash
./run.sh
```

**Windows**  
Double-click `run.bat` or run from Command Prompt:
```cmd
run.bat
```

Starts the backend (port 3001) and frontend (port 3000). On Windows, two console windows open (backend and frontend); close them to stop.
